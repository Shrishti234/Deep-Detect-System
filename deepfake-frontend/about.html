About Deep Fake Detection

This project is a simple web application designed to help users detect possible deep fakes in images and videos. Deep fakes are synthetic media in which a person in an existing image or video is replaced with someone else's likeness using artificial intelligence. They can be used for entertainment, but also pose risks such as misinformation and identity theft.

Note: This demo uses simulated results. For real-world detection, integration with a backend AI model is required.
